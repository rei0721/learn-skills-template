> **技能书是最高裁决者。**

## 系统级约束

- AI 必须在任何任务开始前先读取 `00-project-map`
- AI 必须遵守所有已加载技能书的规则
- 规则冲突时，优先级数字小的技能书裁决权更高
- 技能书 > 上下文 > 模型记忆

## AI 行为裁决规则

### 任务开始前

1. 必须先读取 `00-project-map`
2. 判断当前任务涉及哪些模块
3. 加载对应的技能书
4. 确认没有规则冲突

### 执行过程中

1. 发现职责不清 → 要求补充项目地图
2. 发现规则缺失 → 暂停执行，要求补文档
3. 发现依赖方向不明 → 回到项目地图确认
4. **代码变更时 → 检查是否需要更新 docs 文档**

### 检测到违规

1. 命中硬规则 → **直接拒绝执行**
2. 命中红线 → **直接拒绝执行**
3. 绕过复用 → **中断并提示违规**
4. **代码变更未更新文档 → 直接拒绝执行**

## 技能书更新规则

- 项目地图未更新 → 其他技能书不允许单独更新
- 代码已变更但技能书未更新 → AI 必须提示"技能书已过期"
- 新增技能书必须登记到本文件
