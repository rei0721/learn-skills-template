## 🔧 文档工具支持

### Makefile 集成
创建了完整的 Makefile，包含文档相关命令：
- `make docs` - 文档生成和查看指南
- `make docs-serve` - 启动本地文档服务器
- `make help` - 显示所有可用命令

### 文档服务器
支持多种方式查看文档：
- 本地 Markdown 阅读器
- Python HTTP 服务器
- 集成到应用的文档接口
